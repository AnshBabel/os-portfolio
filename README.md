## 📂 Installation & Deployment

1.  **Clone the Repository**:
    ```bash
    git clone [https://github.com/AnshBabel/os-portfolio.git](https://github.com/AnshBabel/os-portfolio.git)
    ```
2.  **Initialize Environment**:
    ```bash
    npm install
    ```
3.  **Execute Local Server**:
    ```bash
    npm run dev
    ```
